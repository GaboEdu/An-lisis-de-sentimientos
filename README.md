#  Análisis de Sentimientos de Noticias

##  Descripción

Proyecto de análisis de texto que clasifica noticias en **positivas, negativas o neutrales**, aplicando técnicas de procesamiento de datos y lógica de clasificación.

Este proyecto fue desarrollado como práctica de análisis de datos y procesamiento de lenguaje natural, utilizando herramientas de inteligencia artificial.

##  Funcionalidades

* Clasificación de noticias por sentimiento
* Procesamiento de texto
* Análisis automatizado de contenido

## Tecnologías utilizadas

* Python
* Procesamiento de datos
* IA asistida (Claude)

##  Estructura del proyecto

* Scripts de análisis de texto
* Lógica de clasificación
* Manejo de datos de entrada

## Objetivo del proyecto

Desarrollar una solución básica para analizar contenido textual y demostrar habilidades en:

* Procesamiento de datos
* Lógica de programación
* Uso de herramientas de inteligencia artificial

##  Mejoras futuras

* Integrar interfaz web
* Usar APIs de noticias en tiempo real
* Implementar modelos de NLP más avanzados
